Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag B (não semântica)

Nessa frase temos um termo em destaque usando a tag STRONG (semântica)

Itálico / Ênfase

Nesta frase, temos um termo em Itálico usando a tag I ( Não semântica )

Nesta frase, temos um termo em ênfase usando a tag EM ( Semântica )

Texto pequeno

Podemos criar um texto pequeno usando a tag SMALL

Tinhamos a tag <big> para textos grandes, porem foi retirado do HTML5

Texto Deletado

Temos a tag DEL que significa que o texto está ali, pode até ser lido, mas deve ser desconsiderado pelo leitor

Texto Inserido

Essa tag é o oposto do texto deletado.Nesse caso usamos a tag INS para dizer que o texto está ali, deve ser lido e deve presta atenção nele.

Texto Sobreescrito e Subescrito

Usamos sobreescrito para casos que: x2
Usamos subescrito para casos que : h2o

Texto Marcado

Para marcar um texto usamos a tag MARK
Para trocar a cor só com CSS
